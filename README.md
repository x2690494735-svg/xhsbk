# 🔥 小红书热点收集器

小红书热门话题追踪工具，Web 界面操作，开源、轻量、可分享。

## ✨ 功能

- 网页端操作，打开浏览器就能用
- 多关键词搜索，追踪你关心的领域
- 自动采集笔记标题、互动数据、作者信息
- 卡片式展示，按热度排序
- 历史记录随时回顾
- 浏览器状态持久化，登录一次即可持续使用

## 🚀 本地使用（如果你装了 Python）

双击 **启动.bat** → 浏览器自动打开 → 点 **「开始采集」**。

首次运行会自动装依赖；首次采集会弹出浏览器窗口，手动扫码登录小红书。

## 📤 分享给朋友（自动构建 exe）

群友不需要装 Python。流程：

1. 你把代码 push 到 GitHub
2. 打 tag 如 `v1.0.0`，推送触发 GitHub Actions
3. **云端自动打包**，15 分钟左右完成
4. 群友去 Releases 页面下载 `小红书热点收集器.zip`
5. 解压，双击 `启动.bat`，直接能用

这个你只需要做一次——在终端运行：

```bash
git tag v1.0.0
git push origin v1.0.0
```

剩下的 GitHub 全自动搞定。

## ⚙️ 配置说明

网页上可以直接管理关键词。更多配置在 `config.yaml`：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `keywords` | AI, 认知升级, 个人成长 | 监测关键词列表 |
| `crawler.headless` | false | 首次手动登录后改为 true |
| `crawler.max_notes` | 30 | 每个关键词最大条数 |
| `crawler.timeout` | 30 | 页面加载超时（秒） |
